# MFood
Food app using React, .NET Web API, SqlServer Database

<h1> <b>Features</b> </h1>
<h2> BACKEND </h2> 
<ul>
  <li> Full REST API </li>
  <li> CRUD - MongoDB </li>
  <li> Authentication system </li>
</ul>

<h2> FRONTEND </h2> 
<ul>
  <li> Context api  </li>
  <li> Routing </li>
  <li> Profile page </li>
  <li> Friends </li>
  <li> Timeline posts </li>
  <li> Login / Register </li>
  <li> Like / Dislike post </li>
  <li> Upload image </li>
</ul>
